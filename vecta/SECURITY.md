# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vecta, please do **not** open a public GitHub issue.

Email: [security@vecta.io](mailto:security@vecta.io)

PGP key: (add when available)

We aim to respond within 48 hours and provide a fix timeline within 7 days for confirmed issues.

## Scope

**In scope**

- API endpoints at `https://vecta-elaf.onrender.com`
- Landlord portal deployments (e.g. Vecta on Vercel)
- Authentication and token handling
- PII encryption and storage
- Certificate issuance and verification

**Out of scope**

- Social engineering
- Physical attacks
- DoS / DDoS attacks

## Disclosure Policy

We follow coordinated disclosure. Please allow approximately 90 days for a fix before public disclosure, unless otherwise agreed.
